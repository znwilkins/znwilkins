Flood Free Nova Scotia

March 2016.
Website. Created with: HTML, CSS, NodeRed, SQL. Running on IBM Bluemix.

This site was created during the IBM Bluemix/Nova Scotia Open Data Hackathon, in collaboration with three other CS students.

It used the user's location information to ascertain their position within the province, and then determine,
based on ocean rise data and land elevation, when their home could flood if sea levels continue to rise.

Note that the database for this site is no longer operational, so it will not return a result.
Examination of the code does demonstrate how the site used to function.

Robo Curling/Wrestling

December 2015.
Algorithm. Created with: Aseba on Thymio robots.

This program was created for the Fall 2015 Robot Olympics, as the final project of CSCI1107 (Animated Computing).

Using the robot's nine light sensors, and accelerometer, we needed to navigate to the button,
and defend it from another robot with the same objectives.

Star Wars Battlefront II Mods

2006/2007.
Game Modification. Created with: Lua, Softimage XSI, Pandemic's Zero Editor.

I created a number of modifications for Star Wars Battlefront II, especially custom battlefields, and unit modifications.

In this example, I converted a Samus Aran 3D model into a useable character model.
The new model was rigged with a proper skeleton for animation and anchor points for attaching weapons.
